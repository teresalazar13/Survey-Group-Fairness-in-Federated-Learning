### Table 1: Summary of solutions on group fairness in federated learning
- Location (LOC): L - local solutions, G - global solutions, H - hybrid solutions
- Data Partition (D.P.): HFL - horizontal FL, VFL - vertical FL
- Data: TAB - tabular, IMG - image, TXT - text
- Metrics: SP - statistical parity-based, CD - conditioned on the decision, CO - conditioned on the outcome, PB - performance-based, UC - unconditional, LB - loss-based; +DIF - difference, +RAT - ratio, +GS - group-specific, +STD - standard deviation-based, +AVG - average-based, +VAR - variance-based
- Sensitive Attributes (S.A.): SB - single binary, SV - single multivalued, MB - multiple binary
- N.A.: studies that do not propose specific methods but instead focus on conducting experiments related to group fairness
